    # Decode via upload

    Finally, vue-qrcode-reader comes with another component which allows image scanning via classic file upload. If you are on mobile and your browser supports it, your camera automatically starts to take the picture to be uploaded.

    # Source

    <template>
  <div>
    <p class="decode-result">Last result: <b>{{ result }}</b></p>

    <qrcode-capture @decode="onDecode" />
  </div>
</template>

<script>
export default {
  data () {
    return {
      result: ''
    }
  },

  methods: {
    onDecode (result) {
      this.result = result
    }
  }
}
</script>
